## Project Brief / 项目简介

This firmware line is a useful embedded-systems case, but the source files came from a company work folder. The public page keeps the architecture-level description and removes direct source downloads.

这条线适合后续写成完整的嵌入式项目复盘，但当前版本不公开 `.c`、`.ioc`、README、PINOUT 或内部开发文档。

## Firmware Scope / 固件范围

- Periodic temperature and humidity sensing.
- Presence / outlet-state detection.
- Fan PWM control and local OLED state display.
- Bluetooth reporting.
- Flash-backed user setting persistence.
- Separation between generated HAL code, app logic, service modules, and board-support modules.

## Evidence Boundary / 证据边界

Public evidence should later be rewritten as screenshots, logs, diagrams, and masked snippets. Full source files stay private unless the company-sensitive parts are removed.
