## Project Brief / 项目简介

This legacy prototype connects structure, airflow, embedded control, and a desktop-side monitoring workflow. Only a small public evidence set remains because raw design files are not necessary for a public portfolio preview.

这个原型页只保留轻量证据，用来说明项目确实覆盖过结构、风道、控制和桌面端监控。原理图、Gerber、EasyEDA、CAD、3MF、PlatformIO 和 C# 源码暂不公开。

## Reflection / 复盘

The value of this prototype is not polish. The value is the exposed interface between structure, airflow, electronics, software, and documentation. Detailed files can stay in a private archive until they are cleaned and desensitized.
