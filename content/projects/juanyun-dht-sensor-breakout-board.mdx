## Project Brief / 项目简介

This is a smaller Juanyun hardware task, but it belongs in the portfolio because it shows how I think about a simple module as a complete engineering unit: requirement, interface, PCB constraints, firmware reading, and test evidence.

这个项目不是“大板子”，但它适合展示我对小任务的拆解方式。温湿度小板如果只看原理图会很简单，但真正交付时仍然要说明接口、电源、上拉、线长、采样周期、读取失败和固件联调。

## My Work / 我的工作

- 我把 DHT11 / AM2302 小板从卷云大文件夹里单独整理出来，避免它被外机板和底座板资料淹没。
- 我把硬件任务书和底座固件里的双路 DHT 采集逻辑关联起来，让小板设计能回到实际系统使用场景。
- 我把后续需要补的证据列清楚：原理图、PCB、接口定义、上拉电阻选择、读取失败处理、实物测试照片和串口/OLED 显示结果。

## Reflection / 复盘

I want this website to include small engineering tasks as well as large projects. A compact sensor board can still show whether I can define a bounded task, connect hardware and firmware, and leave a clear verification path for the next iteration.

我希望作品集不只放“大而全”的项目。小板任务如果写清楚，也能证明我会把一个具体硬件需求从任务书推进到接口、固件和测试。
