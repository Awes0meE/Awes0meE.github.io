## Project Brief / 项目简介

This page is a public-safe overview of the Juanyun internship archive. The raw source folder contains real company engineering artifacts: board exports, firmware source, datasheets, SOP documents, prototype records, and manufacturing files. Those files are useful for private review, but not suitable for direct public release.

当前公开页不直接提供 Gerber、原理图、BOM、贴片坐标、源码或完整 PDF。页面只保留项目方向、工作范围、板子截图和后续可脱敏补充的位置。需要给导师、HR 或面试官看的版本，可以单独准备脱敏材料。

![AC Unit V2.1 main board render](/uploads/projects/juanyun-tech/acunit-v21-main-front.png)

## Scope / 工作范围

- STM32-based control-board firmware organization.
- AC-unit control-board requirement reading, interface split, and board-version documentation.
- Base-unit sensing, display, fan-control, Bluetooth reporting, and parameter persistence.
- Hardware-development SOP and project handoff documentation.
- Smaller sensor, actuator, cooling-prototype, and motor-control archive lines.

## Public Evidence Boundary / 公开证据边界

For now, the public evidence set is intentionally small. Company-specific production files stay out of the deployed website:

- no Gerber archives;
- no schematic PDFs;
- no BOM or pick-and-place files;
- no EasyEDA/CAD source projects;
- no complete firmware source dumps;
- no internal requirement or manufacturing packages.

## Next Step / 后续补充

Future public material should use desensitized photos, redrawn system diagrams, test-record screenshots, and short technical explanations without company secrets. The goal is to show real engineering participation without publishing the company's complete design files.
