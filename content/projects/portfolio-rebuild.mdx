## Project Brief / 项目简介

The old portfolio only preserved a generated Hexo deployment output. The rebuild turns the site into a maintainable source project with typed content, bilingual pages, versioned assets, and a visual direction closer to an engineering portfolio than a template blog.

旧站的问题不是单个页面不好看，而是没有稳定的源码结构。新版本把内容、页面和资产分开，让项目、学习笔记、媒体材料和后续功能可以逐步增长。

## Scope / 工作范围

- Next.js App Router structure for home, work, notes, media, and about pages.
- File-based MDX content workflow for Git-friendly updates.
- Bilingual public pages with an engineering-academic visual style.
- Vercel deployment and Cloudflare-managed custom domain.

## Next Steps / 后续计划

- Keep replacing placeholder visuals with real project evidence.
- Add stronger content review rules for company and internship materials.
- Extend the site only when the file-based workflow becomes a real bottleneck.
