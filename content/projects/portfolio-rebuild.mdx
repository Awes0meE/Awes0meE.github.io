## Project Brief / 项目简介

The old portfolio was generated from a Hexo tutorial and only preserved deployment output. This rebuild turns it into a maintainable source project with typed content, bilingual pages, and a visual language closer to my engineering identity.

旧站的问题不是单个页面不好看，而是没有可维护的源码结构。新版本会把内容、页面和资产分开，让后续补项目、写笔记、上传媒体都能自然增长。
