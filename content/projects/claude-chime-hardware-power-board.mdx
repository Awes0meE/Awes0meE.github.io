## Project Brief / 项目简述

This page keeps the Claude Chime hardware folder visible as a compact board archive. The source material is small enough to present directly: a board logo image, schematic and PCB PDF exports, a BOM table, Gerber manufacturing archive, EasyEDA project file, and datasheets for the ESP32-S3 and power-stage components.

这页把 Claude Chime 硬件资料作为一个小型板卡档案展示。它不像卷云实习材料那样需要拆很多子线，重点就是单路电磁锁控制器电源管理板：原理图、PCB、BOM、Gerber、EasyEDA 工程和关键器件数据手册都放在下面的公开资料区。

![Claude Chime logo / Claude Chime 标识](/uploads/projects/claude-chime-hardware/claude-chime-logo.png)

## Hardware Shape / 硬件形状

- Controller direction: ESP32-S3 based control and connectivity.
- Power direction: boost / charging / MOSFET switching around a compact control board.
- Output direction: single-channel electromagnetic-lock power management.
- Evidence direction: PDF exports for quick reading, plus Gerber / EasyEDA / BOM files for archive-level inspection.

## Files / 文件

The project-files section below intentionally lists the uploaded files rather than hiding them behind a single download. The PDF and image files can be opened directly; the BOM, Gerber archive, EasyEDA source, and datasheets remain downloadable public evidence.

下方资料区会把上传文件逐项列出来，而不是只放一个“下载全部”。这样后续继续完善项目时，可以在正文里逐个解释：为什么选这个芯片、为什么这样走电源、哪个 PDF 对应哪个阶段、哪些制造文件只是归档证据而不是最终发布包。

## What Stayed Out / 没放上来的东西

The source folder also contains `工程文件密码.txt`; that file is intentionally not copied to the website. Passwords and credentials should stay outside `public/uploads` even when the rest of the board archive is public.

源目录里还有 `工程文件密码.txt`，这个文件没有复制到网站。即使项目资料整体可以公开，密码和凭据也不应该进入 `public/uploads`。
