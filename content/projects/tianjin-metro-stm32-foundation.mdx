## Project Brief / 项目简述

This page archives the STM32 learning path from a Tianjin rail-transit internship folder. The valuable part is not a finished product claim, but the trace of a first real internship: building the toolchain, touching registers, wiring modules, and slowly connecting microcontroller basics with railway control language.

这组材料来自天津轨道交通相关实习。文件夹名是天津津铁通信有限公司，实习报告里记录的方向是轨道交通、OCC、ATC，以及从 STM32 到 PID、再到电机和列车控制系统的学习路径。它更像一次入门训练，不适合包装成“独立负责某系统”，但很适合保留为大学以来第一次实习的真实技术痕迹。

## Source Inventory / 源目录盘点

- `Study_STM32F103C8T6` 是主线材料：基本外设、中断、定时器、PWM、舵机、编码器，以及地铁环境智能检测系统。
- `STM32相关资料` 是学习资料库：Keil5 MDK、参考文档、固件库、模块资料、课件和程序源码，体量很大，很多是 vendor 或网络收集资料。
- `MG995商家资料` 提供舵机控制参数和若干参考例程，适合只抽取控制参数图，不适合整包复制。
- 实习报告 PDF 能帮助确认 2025-02-05 到 2025-02-16 的学习路线，但原文件含学号、联系人和电话，所以没有作为公开下载资产复制。
- 控制中心手册、设备图纸、故障台账、安装包、固件库、`.hex`、`.o`、`.crf`、`.axf` 等文件没有复制到公开目录。

## Learning Route / 学习路线

前几天的任务很基础：先看清 STM32F103C8T6 是什么，Keil MDK 和 ST-Link 怎么连，启动文件、时钟、GPIO 初始化和延时函数怎么形成最小闭环。这个阶段的成就感来自很小的现象：板载 LED 能稳定闪烁，OLED 能显示状态，调试器里能看到寄存器变化。

The learning then moved through GPIO modes, EXTI, timers, PWM, UART, ADC, DMA, SPI, I2C, RTC, watchdog, and FreeRTOS basics. In the folder, those topics appear as separate Keil projects rather than polished notes, which matches the reality of a short placement: learn one peripheral, make it run, keep the trace, then move on.

## Small Integration / 小综合题

最完整的工程是 `x-x STM32地铁环境智能检测系统`。它把 OLED、LDR 光敏传感器、DHT11、编码器、TIM3/TIM4、PWM 调光、三色灯和舵机放在同一个 STM32F103C8T6 工程里。题目有“地铁环境”的包装，但底层训练其实很朴素：传感器读数、定时更新、外部中断输入、PWM 输出、显示刷新，以及如何把几个模块接在一起后还能调得动。

## Evidence Boundary / 证据边界

这次只公开两类证据：一类是 STM32 学习图片，另一类是经过压缩的代码摘录。没有把完整 Keil 工程、商家资料包、固件库、安装程序、地铁控制中心图纸、故障台账或含个人联系信息的报告 PDF 放进 `public/uploads`。

The public page is intentionally a learning archive. It shows the technical path and concrete files without pretending that a short internship produced an industrial rail-control subsystem.

## Review / 复盘

这次实习最重要的东西不是某一个外设，而是第一次把“课程里的单片机”放进真实行业语境里。STM32、PID、电机、ATP/ATO/ATS 这些词在报告里第一次连在一起。现在回头看，有些代码还很入门，有些注释和命名也不成熟，但正因为这样，它能说明当时是在从零搭环境、从寄存器和外设一步一步往系统概念上靠。
