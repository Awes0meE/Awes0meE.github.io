## Project Brief / 项目简介

PID Starter Kit is a reusable control experiment package. The project connects firmware control loops, parameter tuning, serial telemetry, response curves, and desktop-side visualization into one repeatable workflow.

这个项目不是单独写一个 PID 函数，而是把控制实验需要的几件事放在一起：底层控制器、串口协议、调参记录、响应曲线、可视化工具和实验说明。后续接入不同电机、底盘或传感器时，可以沿用同一套记录方式。

## Scope / 工作范围

- Control-loop structure and reusable parameter interface.
- Serial telemetry format for live response observation.
- Step-response, error, and output-curve visualization plan.
- Firmware, desktop tool, and documentation release structure.

## Next Steps / 后续计划

- Add measured response plots and tuning comparisons.
- Record experiment videos and connect them to the media gallery.
- Separate firmware, desktop tool, and documentation into clearer releases.
