## Project Brief / 项目简介

This project involved a product-facing AC unit control platform with multiple boards, compressor and fan interfaces, sensor inputs, UI controls, and firmware-facing constraints. The public page now only describes the engineering scope. The actual schematics, PCB exports, Gerber packages, BOM files, and firmware source are not served from the website.

这条线比普通课程项目更接近公司真实产品，所以公开时必须保守一点。外机板相关的原理图、PCB、生产包、源码和内部需求材料暂时都不放公网。

## My Work / 我的工作范围

- Read and organized hardware requirements around power, sensing, UI, compressor-driver, fan, and EEV interfaces.
- Helped structure the V2.0 / V2.1 board-version story so the project could be explained as a system instead of a single PCB screenshot.
- Connected hardware constraints with firmware bring-up, pin planning, module boundaries, and future test records.

## Public Evidence / 当前公开证据

No raw company design files are public in this validation branch. The next public-safe evidence should be redrawn diagrams, masked board photos, high-level interface tables, and first-person development notes after review.
