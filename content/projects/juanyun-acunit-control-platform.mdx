## Project Brief / 项目简介

The AC unit control platform is a product-facing hardware line with multiple boards, compressor and fan interfaces, sensor inputs, UI controls, and firmware-facing constraints. The public page describes only the engineering scope. Schematics, PCB exports, Gerber packages, BOM files, and firmware source are not served from the website.

这条线比普通课程项目更接近公司真实产品，公开时需要保守处理。外机板相关的原理图、PCB、生产包、源码和内部需求材料暂不放公网。

![AC Unit V2.1 system block diagram](/uploads/projects/juanyun-tech/acunit-v21-system-block.png)

## Scope / 工作范围

- Hardware requirements around power, sensing, UI, compressor-driver, fan, and electronic-expansion-valve interfaces.
- V2.0 / V2.1 board-version structure explained as a system instead of a single PCB screenshot.
- Hardware constraints connected with firmware bring-up, pin planning, module boundaries, and future test records.
